About this site

l3ms is a lightweight learning management system. It provides a gradebook, account management, and authentication for static pages and software repositories. It was developed by by Christopher League using Python and the Django web framework. The source is available on github.

liucs.net hosts several courses and other resources from the Computer Science department at Long Island University's Brooklyn Campus.
